# Assignment 2 Report



# Step 1: Create a GitHub Pages Repository

1. Open the GitHub website:
   https://github.com

2. Log in to the GitHub account.

3. Click the **"+"** icon at the top-right corner.

4. Select **New repository**.

5. Create a new repository with the following settings:

| Setting         | Value                     |
| --------------- | ------------------------- |
| Repository name | `your-username.github.io` |
| Visibility      | Public                    |

Example:

```text id="m2mjlwm"
phucthanh.github.io
```

6. Check:

*  Add a README file

7. Click **Create repository**.

---

# Step 2: Install GitHub Desktop

1. Download GitHub Desktop from:
   https://desktop.github.com/

2. Install the application.

3. Log in using the GitHub account.

---

# Step 3: Clone Repository Using GitHub Desktop

1. Open GitHub Desktop.

2. Select:

```text id="2h4tl7"
File → Clone Repository
```

3. Choose the repository:

```text id="bdjz7o"
your-username.github.io
```

4. Select a local folder to save the project.

5. Click **Clone**.

---

# Step 4: Download the Jekyll Theme

1. Open the YAT theme page:
   http://jekyllthemes.org/themes/jekyll-theme-yat/

2. Open the GitHub repository of the theme.

3. Download the theme source code as ZIP.

4. Extract the ZIP file.

5. Copy all extracted files into the cloned repository folder.

6. Replace existing files if necessary.

---

# Step 5: Edit the Website

1. Open the repository folder using Visual Studio Code.

2. Modify the website information such as:

* Name
* Introduction
* Resume content
* Skills
* Projects
* Contact information

3. Save all edited files.

---

# Step 6: Commit Changes Using GitHub Desktop

1. Return to GitHub Desktop.

2. GitHub Desktop automatically detects changed files.

3. In the bottom-left corner, enter a commit message:

```text id="uwcqgn"
Add GitHub Pages website and resume
```

4. Click:

```text id="h7js0y"
Commit to main
```

---

# Step 7: Push Files to GitHub

1. After committing changes, click:

```text id="kngn4j"
Push origin
```

2. Wait for the upload process to complete.

---
